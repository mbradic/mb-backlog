Precondition:

Application is open with the task list displayed and contains at least one task

Test Steps:
Step 1: Navigate to the tasks view
View wireframe for task list
Step 2: Click on a task to open its details
View wireframe for task details view
Step 3: Observe the task details view is opened
View wireframe for task details view
Expected Result:

A new view or modal opens displaying the complete details of the selected task.
