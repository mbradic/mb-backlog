TC-021: Edit task state

Precondition:

Application is open with an existing task in "To Do" state and edit functionality is available

Test Steps:
Step 1: Open task details or task list and select Edit
View wireframe for edit task form
Step 2: Change state from "To Do" to "In Progress"
View wireframe for edit task form with state changed
Step 3: Click Save button
View wireframe for updated task list
Expected Result:

Task state is updated successfully. The task list shows the new state.
