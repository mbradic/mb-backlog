mb-backlog - FR-006 - Edit task - 1.0

Description: User can edit all fields of an existing task (code, title, state, and description). The edit functionality should allow the user to modify any task field and save the changes. The application should validate that all required fields remain populated and that the task code remains unique if changed.

Related User Role: Owner

Manual Testing Scenarios
  • TC-020: Edit task title
  • TC-021: Edit task state
  • TC-022: Edit task description
  • TC-023: Edit task code to new unique value
  • TC-024: Attempt to edit task with invalid data
