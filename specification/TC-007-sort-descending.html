TC-007: Sort tasks in descending order

Precondition:

Application is open with tasks already sorted in ascending order by a column

Test Steps:
Step 1: Navigate to the tasks view with ascending sort
View wireframe for sorted task list
Step 2: Click on the currently sorted column header again
View wireframe for reverse sorted task list
Step 3: Verify tasks are now sorted in descending order
View wireframe for reverse sorted task list
Expected Result:

Tasks are now sorted in descending order. The sort indicator changes to show descending direction.
