mb-backlog - FR-007 - Delete task - 1.0

Description: User can delete an existing task from the backlog. The delete functionality should remove the task permanently from the system. The user should be prompted to confirm the deletion to prevent accidental removal of tasks.

Related User Role: Owner

Manual Testing Scenarios
  • TC-025: Delete task after confirming deletion
  • TC-026: Cancel task deletion
  • TC-027: Verify deleted task no longer appears in task list
  • TC-028: Verify confirmation prompt is shown before deletion
