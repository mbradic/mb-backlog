TC-003: Verify all task fields are displayed

Precondition:

Application is open with at least one task containing all fields (code, title, state)

Test Steps:
Step 1: Navigate to the tasks view and locate a task
View wireframe for task list with fields
Step 2: Verify that columns for code, title, and state are visible
View wireframe for task list with fields
Expected Result:

All required fields (code, title, state) are displayed as columns in the task list. Each task row shows values for all required fields.
