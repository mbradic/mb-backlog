TC-001: View tasks list when no tasks exist

Precondition:

Application is open and all tasks have been deleted or this is a fresh installation with no tasks

Test Steps:
Step 1: Navigate to the tasks view
View wireframe for empty task list
Step 2: Observe the display for any tasks
View wireframe for empty task list
Expected Result:

The task list displays with an empty state message indicating that no tasks exist. The interface remains functional with options to create a new task.
