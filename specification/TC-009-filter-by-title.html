TC-009: Filter tasks by title

Precondition:

Application is open with multiple tasks having different titles

Test Steps:
Step 1: Navigate to the tasks view
View wireframe for filter panel
Step 2: Open the filter panel and enter text in the title filter field
View wireframe for filtered task list
Step 3: Verify only tasks with matching titles are displayed
View wireframe for filtered task list
Expected Result:

Only tasks whose titles contain the entered text are displayed. The filter is case-insensitive and matches partial titles.
