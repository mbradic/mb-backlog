TC-004: Sort tasks by code column

Precondition:

Application is open with multiple tasks with varying task codes (e.g., TASK-001, TASK-003, TASK-002)

Test Steps:
Step 1: Navigate to the tasks view
View wireframe for task list
Step 2: Click on the "Code" column header to sort
View wireframe for sorted task list
Step 3: Verify the tasks are now sorted by code
View wireframe for sorted task list
Expected Result:

Tasks are sorted in ascending order by code. The sort indicator (arrow or similar) appears next to the Code column header.
