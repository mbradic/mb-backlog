Precondition:

Application is open with multiple tasks in different states and with different titles

Test Steps:
Step 1: Navigate to the tasks view
View wireframe for filter panel
Step 2: Set filter for state = "Done" and title contains "urgent"
View wireframe for multi-filtered task list
Step 3: Verify only tasks matching both filters are displayed
View wireframe for multi-filtered task list

Only tasks that match ALL active filters are displayed. Tasks must be in "Done" state AND have "urgent" in their title.
