mb-backlog - FR-002 - Sort tasks - 1.0

Description: User can sort tasks by a single column at a time. The sort functionality should allow sorting in ascending and descending order. Supported columns include code, title, and state. The current sort order should be clearly indicated in the user interface.

Related User Role: Owner

Manual Testing Scenarios
  • TC-004: Sort tasks by code column
  • TC-005: Sort tasks by title column
  • TC-006: Sort tasks by state column
  • TC-007: Sort tasks in descending order
