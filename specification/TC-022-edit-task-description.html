TC-022: Edit task description

Precondition:

Application is open with an existing task that has or doesn't have a description and edit functionality is available

Test Steps:
Step 1: Open task details or task list and select Edit
View wireframe for edit task form
Step 2: Modify or add description text
View wireframe for edit task form with description
Step 3: Click Save button
View wireframe for updated task list
Expected Result:

Task description is updated successfully and saved. The description is accessible in the task details view.
