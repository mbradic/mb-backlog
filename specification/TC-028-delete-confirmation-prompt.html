TC-028: Verify confirmation prompt is shown before deletion

Precondition:

Application is open with an existing task and delete functionality is available

Test Steps:
Step 1: Select a task and click Delete button
View wireframe for delete confirmation dialog
Step 2: Observe if a confirmation dialog appears
View wireframe for delete confirmation dialog
Step 3: Verify the dialog includes task information and confirm/cancel buttons
View wireframe for delete confirmation dialog
Expected Result:

A confirmation prompt is displayed before deletion. The prompt shows the task being deleted and requires explicit confirmation before proceeding with the deletion.
