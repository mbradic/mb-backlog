TC-005: Sort tasks by title column

Precondition:

Application is open with multiple tasks with varying titles

Test Steps:
Step 1: Navigate to the tasks view
View wireframe for task list
Step 2: Click on the "Title" column header to sort
View wireframe for sorted task list
Step 3: Verify tasks are sorted alphabetically by title
View wireframe for sorted task list
Expected Result:

Tasks are sorted in ascending alphabetical order by title. The sort indicator appears next to the Title column header.
