TC-006: Sort tasks by state column

Precondition:

Application is open with multiple tasks having different states (To Do, In Progress, Done)

Test Steps:
Step 1: Navigate to the tasks view
View wireframe for task list
Step 2: Click on the "State" column header to sort
View wireframe for sorted task list
Step 3: Verify tasks are sorted by state
View wireframe for sorted task list
Expected Result:

Tasks are sorted in ascending order by state. The sort indicator appears next to the State column header.
