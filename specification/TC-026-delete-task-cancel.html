Precondition:

Application is open with an existing task and delete functionality is available

Test Steps:
Step 1: Select a task and click Delete button
View wireframe for delete confirmation dialog
Step 2: Confirmation dialog appears, click "Cancel" button
View wireframe for delete confirmation dialog
Step 3: Verify dialog closes and task remains in list
View wireframe for task list
Expected Result:

Deletion is cancelled and the task remains in the task list unchanged.
