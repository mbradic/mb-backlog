mb-backlog - FR-005 - Create task - 1.0

Description: User can create a new task by providing all required fields (code, title, and state) and optionally some or all optional fields (description). After providing the necessary information, the user should be able to save the new task to the backlog. The application should validate that all required fields are provided and that the task code is unique.

Related User Role: Owner

Manual Testing Scenarios
  • TC-016: Create task with only required fields
  • TC-017: Create task with all fields including optional description
  • TC-018: Attempt to create task with missing required field
  • TC-019: Attempt to create task with duplicate code
